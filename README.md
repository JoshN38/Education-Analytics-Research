Analyzing Analytics in Education
Project Overview

This project is an academic research analysis that explores how data analytics and predictive modeling are applied in the education sector to improve student success, graduation rates, and institutional decision-making. The analysis is based on published research and real-world case studies and was completed as part of a Management Information Systems (MIS) course.

Project Objectives

Examine how analytics is used in educational institutions

Understand the role of predictive analytics in improving student outcomes

Evaluate benefits, limitations, and ethical considerations

Review real-world implementations of education analytics

Case Study
Georgia State University

Reviewed the use of predictive analytics and data-driven advising systems

Analyzed early-alert models for identifying at-risk students

Examined reported improvements in graduation rates and time-to-degree

Key Topics Covered

Predictive analytics in higher education

Education Data Mining (EDM)

Student performance and retention analysis

Data privacy, bias, and model accuracy

Key Takeaways

Analytics can improve educational outcomes when applied responsibly

Early intervention models help reduce dropout rates

Ethical and accurate use of student data is essential
