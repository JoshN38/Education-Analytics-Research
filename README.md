Analyzing Analytics in Education
Overview

This project is a school-based research analysis examining how data analytics and predictive modeling are applied in the education sector to improve student success, graduation rates, and institutional decision-making.

The project reviews real-world examples and academic sources to evaluate both the benefits and limitations of using analytics in education.

Topics Covered

Use of analytics to improve student performance and retention

Predictive analytics for identifying at-risk students

Education Data Mining (EDM) techniques

Ethical concerns including data privacy and model accuracy

Case Study Highlight

Georgia State University

Reviewed the use of predictive analytics and data-driven advising systems

Analyzed how early alerts and targeted advising improved graduation rates and reduced time to degree completion

Key Takeaways

Analytics can positively impact student outcomes when used responsibly

Predictive models help institutions intervene earlier

Data privacy, bias, and accuracy remain important concerns

Project Type

Academic research project completed as part of a Management Information Systems (MIS) course.
